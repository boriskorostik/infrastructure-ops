# Дурак Онлайн

Мультиплеерная браузерная игра в карты "Дурак" для 2-6 игроков. Один игрок создает комнату, остальные подключаются по коду через браузер.

## Запуск

Если на сервере уже есть Node.js 18+:

```bash
npm install
npm start
```

После запуска откройте:

```text
http://localhost:3000
```

Чтобы игроки подключались с других устройств, откройте им адрес вашего сервера:

```text
http://IP_ВАШЕГО_СЕРВЕРА:3000
```

Если сервер находится за роутером, пробросьте TCP-порт `3000` на машину с игрой.

## Запуск через Docker

Если Node.js/npm на сервере не установлены:

```bash
docker build -t durak-online .
docker run --rm -p 3000:3000 --name durak-online durak-online
```

## Правила в этой версии

- Колода 36 карт.
- От 2 до 6 игроков.
- Первый ходит игрок с младшим козырем.
- Защитник покрывает карты старшей картой той же масти или козырем.
- Атакующий может подкидывать только ранги, которые уже есть на столе.
- После успешной защиты атакующий нажимает "Отбой".
- Защитник может нажать "Взять".

Это базовая версия подкидного "Дурака"; ее можно развить дальше: добавить зрителей, чат, повторное подключение, переводного дурака и постоянные аккаунты.
